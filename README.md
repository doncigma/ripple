This is a prototype of Ripple, a miniplayer that captures your current media and provides basic control over it to minimize your workflow. 
If you like to have media open while you're working, whether it be music, videos, podcasts, or audiobooks, if it integrates with the APIs Ripple uses, then you'll be able to finally minimize those windows and focus. 
It's in the background, a tiny ripple in your workflow.

To use:
  * Simply copy the repo, navigate to the bin/ folder and click ripple.exe.
